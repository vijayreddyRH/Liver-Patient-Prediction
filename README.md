🩺 Liver Patient Prediction using Machine Learning

This project aims to predict whether a patient is likely to have liver disease using machine learning classification techniques. The model is built using patient medical records and biochemical parameters to assist in early disease detection and support clinical decision-making.

📌 Project Objectives
Perform Exploratory Data Analysis (EDA) to understand the dataset.
Clean and preprocess medical data.
Handle missing values and encode categorical features.
Train and compare multiple machine learning classification models.
Evaluate model performance and identify the best predictive model.
